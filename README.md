# Calculatrice Simple — React + Vite

TP 3.1 : simulateur de calculatrice simple développé en React (avec Vite).

L'application permet de saisir deux nombres, de choisir une opération mathématique
et d'afficher le résultat.

## Fonctionnalités

- Opérations de base : addition, soustraction, multiplication, division.
- **Fonctions avancées** : puissance, modulo, racine carrée, pourcentage.
- **Validation des entrées** : messages d'erreur si un nombre est manquant,
  division/modulo par zéro, ou racine carrée d'un nombre négatif.
- **Historique des calculs** effectués pendant la session, avec bouton pour l'effacer.

## Lancer le projet

```bash
npm install
npm run dev
```

Ouvrez ensuite l'adresse affichée dans le terminal ( http://localhost:5173).
